# -FX-whitelist
Whitelist script for FX Server

## Getting Started
1) Import .sql in your database

2) Add whitelist folder into resource 



3) Add in server.cfg

```
start whitelist
```

## How to use 

Simple add steamid in whitelist table , you don't need to reboot the server 

```
steam:110000106XXXXXf6e
```
